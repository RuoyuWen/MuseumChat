========================================
  Troubleshooting
========================================

Q: "npm is not recognized as an internal or external command"?
A: Please install Node.js (version 18+)
   Download: https://nodejs.org/

Q: Port already in use?
A: Run "stop.bat" first, then start again

Q: Browser didn't open automatically?
A: Manually visit http://localhost:5173

Q: API call failed?
A: Check if API Key is correct and has sufficient balance
