# 快速启动指南

## 前置要求

- Node.js 18+ 
- npm 或 yarn
- OpenAI API Key

## 安装步骤

1. **安装所有依赖**
   ```bash
   npm run install:all
   ```

2. **启动开发服务器**
   ```bash
   npm run dev
   ```
   
   这将同时启动：
   - 后端服务器：http://localhost:3000
   - 前端应用：http://localhost:5173

## 使用流程

1. **打开浏览器**
   - 访问 http://localhost:5173

2. **配置设置**
   - 输入你的 OpenAI API Key
   - 选择AI模型（默认：GPT-4 Turbo）
   - 输入文物的历史背景信息
   - （可选）展开"Prompt 自定义设置"来调整各角色的提示词

3. **开始对话**
   - 点击"开始导览"按钮
   - 在聊天界面输入你的问题
   - 系统会自动选择合适的AI角色回复
   - 如果有多个角色需要回复，会出现"继续"按钮

## 功能说明

### 三个AI角色

- **文物**：以第一人称描述文物的历史、外观、材质等
- **作者**：分享创作背景、制作过程、艺术理念
- **导览员**：提供专业的导览和解释

### 对话管理器

系统会自动分析用户的问题，选择合适的角色回复。例如：
- "请介绍一下这个文物的历史背景" → 可能先由作者介绍，然后文物补充
- "这个文物是什么材质的？" → 由文物本身回答
- "能帮我总结一下吗？" → 由导览员总结

### Prompt自定义

你可以在首页自定义每个角色的Prompt，让AI更好地扮演特定角色。

## 故障排除

### 后端无法启动
- 检查端口3000是否被占用
- 确保已安装所有依赖：`cd server && npm install`

### 前端无法启动
- 检查端口5173是否被占用
- 确保已安装所有依赖：`cd client && npm install`

### API调用失败
- 检查API Key是否正确
- 确认API Key有足够的余额
- 检查网络连接

## 生产部署

1. **构建项目**
   ```bash
   npm run build
   ```

2. **启动生产服务器**
   ```bash
   cd server
   npm start
   ```

3. **部署前端**
   - 将 `client/dist` 目录部署到静态文件服务器
   - 或使用 `npm run preview` 预览构建结果
